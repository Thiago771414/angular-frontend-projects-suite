# Overview

This repository contains a **suite of practical frontend applications built with Angular**.

The project demonstrates the implementation of several frontend engineering concepts including:

- component-based architecture
- services and dependency injection
- routing and navigation
- reactive forms and validation
- HTTP API integration
- dashboard interfaces
- reusable UI components

The repository includes **7 practical Angular applications**, each focused on different frontend development scenarios.

---

# Case Study

## Problem

Modern frontend applications require:

- modular architecture
- reusable components
- structured routing
- dynamic UI updates
- integration with external APIs

Frameworks like Angular help developers build scalable applications using structured patterns.

# Architecture

Angular applications follow a **component-based architecture**.
```text
User Interface
│
▼
Angular Components
│
▼
Angular Services
│
▼
HTTP / External APIs
```

Core Angular concepts used in this project:

- Components
- Services
- Pipes
- Directives
- Routing
- Observables
